URL Shortener Microservice

Usage

To create a shortened URL append /new/ followed by the URL you want to shorten:

https://frozen-dusk-72112.herokuapp.com/new/https://www.google.com

You will receive a JSON response with the original and shortened URL:

{"original_url":"https://www.google.com", "short_url":"https://frozen-dusk-72112.herokuapp.com/1" }

Using the shortened URL will direct you to the original page.
